# WorkIndia InShorts Backend project

## Prerequisite
1. Node version 18.17
2. npm version 9.6.7
3. mysql version 14.14 
   
## How to run the code
1. Clone the repo
2. cd WorkIndia_InShorts
3. Update DB username, password, IP and database in .env and start mysql server
4. npm install
5. node index.js
